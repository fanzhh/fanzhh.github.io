2021年读书记录

Posted on 2022-01-09 Edited on 2022-09-18 In 读书笔记

2021年读过的书。

不知道别人读书的动力是什么，我读书的动力始终来自好奇——好奇世界为何如此、生命为何如此，好奇历史上发生过什么，好奇别人眼中的世界是怎样的体验，还想知道世界的本质，生命的意义。感谢这些书，陪我度过无数旅途与夜晚，我知道的更多，又似乎困惑并没有减少。总之我相信今是昨非，永远期待新的知识颠覆我既有的三观。

科普

这几年我的科普类数目受土摩托影响很大。今年读了他写的《人类的终极问题》，探讨人类的来源、变老的奥秘以及人类的创造力。尼克·莱恩的《复杂生命的起源》也是土摩托强力推荐的一本书，讨论在40亿年的漫长岁月中，从简单的细菌到令人敬畏的复杂生命，这样的演化飞跃事件为何只发生了一次。几年前还读过尼克·莱恩《生命的跃升》，讲进化论的十大伟大“发明”，也强烈推荐。《深海浅说》也是土摩托推荐，汪品先院士所写，有识有趣有干货，典型的大家小书，强烈推荐。其他还读了《源自尘埃的世界 : 元素周期表如何塑造生命》，以元素周期表为线索讨论生命由来。《解密 Instagram : 一款拍照软件如何改变社交》讲Instagram的成长史，特别建议年轻人读，看看人类是如何被算法、资本控制情绪和窃取、时间、金钱乃至生命的。《适者降临 : 自然如何创新》讲进化论，我这几年已经成为进化论的深度痴迷和崇拜者。《鳗鱼的旅行 : 一场对目标与意义的探寻》介于科普与散文之间，探寻鳗鱼之谜同时追忆父子之情，尤其好。《章鱼的心灵》《鱼的好奇心》这两本书是因为《鳗鱼的旅行》和奥斯卡最佳纪录片《我的章鱼朋友》的影响，对鱼类的好奇一发不可收。我们对其他生命了解的太少太少。对了，有强有力的证据表明，一些无脊椎动物包括章鱼、龙虾、螃蟹是有知觉的，能够感觉到疼痛，英国已计划将其列入《动物福利法》。《瘟疫与人》显然是因为这次新冠疫情读的，人类的历史就是瘟疫的历史啊。我关注的好几个人都推荐了大卫·爱登堡的自传，这老爷子的人生才称得上是有趣充实的一生啊。

    哲学和心理学

    不要被《蛤蟆先生去看心理医生》书名误解，这并不是本写给孩子看的书，而是非常有深度的心理疗愈读物。《被讨厌的勇气 : “自我启发之父”阿德勒的哲学课》以对话的方式讲解阿德勒的哲学观点，这本书我记了很多笔记，阿德勒的很多观点如心理创伤并不存在、愤怒都是捏造的、放下别人等与我一直以来的想法产生共鸣。《世俗主义之乐 : 我们当下如何生活》是我2021年读到的最重要的一本书，做了很多笔记还没整理，这本书的观点是我前半生领悟的总结，后半生我也将继续遵循。哲学和心理学有一天会不会被科学替代，我相信有可能，而且希望越早越好。《中国国家治理的制度逻辑 : 一个组织学研究》不宜做过多评价，只能说这个国家很多周而复始的怪相和悲剧这本书能找到答案。

    历史

    埃及是史上最古老的文明，公元前3000年第一位法老就在埃及建立了统治，对古埃及的好奇心一直都有。忘记了读《太平天国革命运动史》的原因了，但真的刷新了我对天平天国的认知。阅读中间数次掩卷叹息：我们这个民族，一百多年前与转型成为现代文明国家擦肩而过，可惜！而洪仁轩可谓中华千古年来第一人。
  • 《破解古埃及》[英]莱斯利、[英]罗伊·亚京斯 / 黄中宪 / 生活·读书·新知三联书店
  • 《太平天国革命运动史》简又文 / 王然 / 后浪丨九州出版社

    小说

    影响我阅读书目的还有《读库》的几个编辑，通常在她们的公众号上看到后，如果适合我的口味我会购买，偶尔直接盲盒下单，读后一般都不差，《你的夏天还好吗》《钦天监》都是。《失踪者》是卡夫卡一篇未竟的小说，全书像是梦游者的呓语，又像精神病患者的自白，读来感同身受。《两京十五日》是马伯庸的又一部畅销书，剧情满分，据说日本在筹拍剧集了。《我爱迪克》是2021年读的最后一本书，看前三分之一差点忍不住放弃，以为又是一部发达国家中产阶级文人的无病呻吟，后来被吸引，被感动。

  • 《钦天监》西西 / 世界·观 广西师范大学出版社

    其他

    何伟（彼得·海斯勒）不是冰冷的旁观者，是有温度的记录者，而且常常直接影响他的记录对象。这也许有悖记者的中立身份，却让我们看到一个温暖善良的人。书中的有些矛盾是由于中西文明的冲突，更多的是因为文明与野蛮的冲突、先进与野蛮的冲突。哦，何伟2021年被四川大学解约了。作为严肃知识分子的刘擎成为网红，也是一个怪现象。这当然是好事，他的书越多人读这个世界会更好一点点。读畅销书《每周工作4小时》是因为读过作者另一部我觉得还不错的书《tools of titans》。读过罗新的《从大都到上都》后就特别喜欢这样的历史大散文，《土星之环》属于此类，这样的书漫无边际又妙趣横生，看似散漫其实要求作者有极深的知识沉淀。
