业余时间做的项目

食堂打卡

使用Django + 微信小程序开发。控制就餐人员的就餐时间和频次。

值班备勤管理

使用Django开发。在线填报值班备勤人员。

某地创建食品安全市督导记录

使用Django + React开发。创建食品安全市督导检查记录、整改情况跟踪。

利用信息技术学习高中数学

使用React开发。高中数学课本中有信息技术内容，许多学校可能因为上机不方便所以略过不讲，但是信息技术可以帮助理解抽象的数学知识。

鲁教2013版五四学制初中英语单词同步测试练习

使用Django Rest + React开发。为了强化孩子单词词汇量迎接中考同时为了熟悉React.js而做的。

在线错题本

Django Rest + React + redux。网上记错题，基于知识点管理错题。

食品药品监管知识在线题库

为练习React.js而做的。在herokuapp上，访问速度较慢。

药品生产单位管理系统

基于python的web.py框架开发。含药品生产单位基础信息管理，检查报告上传分析，检查缺陷汇总，缺陷对应条款的（基于机器学习）的自动生成等功能。

药品GMP缺陷对应条款智能判断机器人

基于python的django框架开发。这是学习机器学习的练习。根据输入的药品生产单位不符合《药品生产质量管理规范》的缺陷描述，智能判断对应的缺陷条款。
