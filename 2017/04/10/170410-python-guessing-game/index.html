猜数游戏[难度:低]

Posted on 2017-04-10 Edited on 2022-01-08 In 技术笔记

这是一个使用python编写的猜数游戏，游戏中会请用户猜测1-99之间的一个数字。

这个游戏的代码使用python的random库产生一个1-99之间的随机数字。代码中还包括一个while循环，一直到用户猜中数字循环才会退出。代码中还使用了if、elif条件判断语句。
