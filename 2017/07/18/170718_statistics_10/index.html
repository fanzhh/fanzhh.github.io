统计学与pandas学习（十）

Posted on 2017-07-18 Edited on 2022-01-08 In 技术笔记

第十章《以测定温度为例，探寻95%置信区间——区间估计》。

总结：

  • 区间估计是这样一种估计方法：它针对母群体的总体参数（参数），在假定其总体参数的情况下，只集合了现实观测到的数据在观测数据的“95%预测命中区间”的总体参数。根据区间估计确定的总体参数的范围叫作“95%的置信区间”。
  • 由区间估计求得的区间，是对所有的总体参数进行“检验”的操作，不舍弃而保留下的集合。
  • 关于正态母群体已知标准差a时，对未知的平均值u进行区间估计的方法

    使用观测到的数据x，解关于u的一元一次不等式
    -1.96 <= (x-u)/a <= +1.96
    得出“ <= u <= ”的形式即可。

  • 95%置信区间是这样一种区间：它由各种各样的观测值用相同方法进行区间估计，其中95%包含正确的总体参数。

练习

已知血压检查中，根据检查者的习惯和听觉，一定程度上会有各种结果。
现在，将自己进行血压测量时的血压设为x，将x作为实际血压平均值为u、标准差为a的正态分布。
此时，如果计测的血压为130,可推测你的实际血压u在哪个范围？请求95%的置信区间。

解：
-1.96 <= ((130)-u)/(6) <= +1.96得
-11.76 <= (130)-u <= 11.76
-141.76 <= -u <= -118.24
118.24 <= u <= 141.76
这是u的95%置信区间。
